What this upload page does

This is a server-backed upload page for sharing images by link. Unlike local utilities such as the JSON, diff, or image resize tools, this page sends the selected file to the Mini-Tools upload endpoint so that a hosted URL can be created. That makes the feature useful for temporary sharing, but it also means the page should explain retention and storage honestly.

Is this local-only?

No. This page uploads the file to a remote service so that a shareable image URL can be created.

What does the retention setting do?

It tells the upload service how long the image should be kept before deletion is scheduled.

What is the storage code for?

It is an optional access code for approved long-term storage. Normal uploads can leave it blank.

Why is there anti-bot verification?

It helps reduce spam and automated abuse before the upload request is sent to the service.
