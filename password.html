How this password generator stays private

This page generates passwords directly in your browser using the Web Crypto API. That means the random values come from your device, not from our server, and the generated password is never uploaded. It is a straightforward local utility: choose your settings, generate a password, copy it, and store it safely in a password manager.

Is it private?

Yes. Password generation happens locally in your browser and nothing is sent to our servers.

Does it include every selected character type?

Yes. If you select uppercase, lowercase, numbers, or symbols, the final password includes at least one character from each selected group.

How long should the password be?

For important accounts, a longer password is usually better. Sixteen characters or more is a sensible baseline.

Should I copy it to the clipboard?

You can, but it is best to paste it straight into a trusted password manager and avoid leaving it there longer than needed.
